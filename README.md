# 🎓 PamplonAI - Asistente Académico Inteligente

**PamplonAI** es un asistente inteligente diseñado para la comunidad de la **Universidad de Pamplona**. Utiliza IA de vanguardia para resolver dudas sobre reglamentos, costos de matrícula, calendarios académicos y ubicaciones de sedes en tiempo real.

## 🚀 Características
- **🧠 Cerebro RAG**: Consulta el reglamento estudiantil y calendarios oficiales (2026-1).
- **🌐 Web Scraping**: Noticias e información de inscripciones en tiempo real.
- **💰 Calculadora de Costos**: Desglose detallado por programa y estrato socioeconómico.
- **📍 Guía de Sedes**: Información completa de Pamplona, Villa del Rosario y Cúcuta.

## 🛠️ Stack
- **Backend**: FastAPI (Python)
- **IA**: Groq / Llama-3
- **Frontend**: Vanilla JS & CSS (Technical Light Design)
